# Bubble Pop

A simple tap-to-pop arcade game for mobile browsers.

## How to Run
1. Open `index.html` in a mobile browser, or
2. Wrap in an Android WebView for app distribution

## Gameplay
- Colorful bubbles rise from the bottom of the screen
- Tap bubbles to pop them and earn points
- Don't let bubbles escape off the top — you only have 3 lives
- Game gets faster as your score increases
- Chain quick pops for combo bonus points

## Controls
- **Tap/Click**: Pop a bubble
- **Start button**: Begin or restart the game
