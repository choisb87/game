# Market Benchmark: Orbit Shield

## 경쟁작 분석

### 1. Orbia: Tap and Relax
- **장르**: 궤도 기반 타이밍
- **장점**: 미니멀 비주얼, 편안한 분위기
- **단점**: 너무 느린 템포, 긴장감 부족
- **참고점**: 원형 궤도 UI의 직관성 확인

### 2. Duet Game
- **장르**: 원형 회피
- **장점**: 미니멀+하드코어 밸런스, 음악 연동
- **단점**: 학습 곡선이 가파름
- **참고점**: 원형 조작의 만족감, 실패 피드백

### 3. Geometry Dash
- **장점**: "한 번 더" 유발, 완벽한 리듬 설계
- **단점**: 과도한 암기 요소
- **참고점**: 즉사 메카닉의 긴장감 → Orbit Shield은 3라이프로 완화

### 4. Super Hexagon
- **장점**: 극한의 집중력, 짧은 세션
- **단점**: 진입 장벽 높음
- **참고점**: 회전 기반 메카닉의 중독성

### 5. One More Dash
- **장점**: 단순한 탭 조작, 깔끔한 비주얼
- **단점**: 반복적, 깊이 부족
- **참고점**: 원탭 인터페이스의 접근성

## 포지셔닝 맵

```
복잡도 높음
    │

    │
    │  Orbia
복잡도 낮음 ──────────────────── 전략 깊이 높음
```

## 채택할 요소
- **원탭 조작**: Orbia/One More Dash의 접근성
- **콤보 보상**: Geometry Dash의 "연속 성공" 쾌감
- **점진적 난이도**: Super Hexagon의 압박감 (단, 3라이프로 완충)
- **시각 피드백**: 파티클 폭발, 화면 흔들림, 슬로우 모션
- **프리미엄 품질감**: 광고 없는 깔끔한 경험

## 피할 요소
- **암기 기반 난이도**: 순수 반사신경 + 판단력 기반으로
- **과도한 난이도**: 첫 판에서도 30초 이상 생존 가능하도록
- **복잡한 튜토리얼**: "탭하면 방향 전환" — 설명 끝
- **IAP/광고 유혹**: 프리미엄 결제 후 완전한 경험
- **단조로운 비주얼**: 우주 테마 + 네온 이펙트로 시각적 다양성

## 경쟁 우위
1. **독특한 메카닉**: "궤도 방패" — 기존 원형 게임과 차별화된 공격 방어 역전
2. **깊은 콤보 시스템**: 최대 x10 배율 + 타이머 압박
3. **다중 적 유형**: 1~3 HP 소행성으로 전략적 깊이
4. **만족스러운 피드백**: 튕겨내기 파티클 + 화면 흔들림 + 슬로우 모션
5. **적정 난이도 곡선**: 처음엔 쉽고, 레벨 10+에서 고수 도전
