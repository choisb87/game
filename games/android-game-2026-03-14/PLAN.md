# Orbit Shield — 게임 디자인 플랜

## 핵심 컨셉
원형 궤도 방어 게임. 행성 주위를 도는 방패로 소행성을 튕겨내는 단순하면서도 중독성 있는 메카닉.

## 30초 체험 시나리오
1. **0-3초**: 탭하면 방패가 회전 방향 전환. 즉시 조작감 학습
2. **3-8초**: 첫 소행성 도착. 방패를 맞춰서 튕겨냄 → 파편 이펙트 + 점수 팝업
3. **8-15초**: 소행성 2-3개 동시 접근. 빠른 판단으로 방향 전환 타이밍 조절
4. **15-25초**: 콤보 x3 달성. 황금빛 콤보 표시 + 점수 배율 체감
5. **25-30초**: 난이도 상승. 중장갑 소행성 등장. 2번 맞춰야 파괴 → 새로운 전략적 깊이

## 판단 레이어 (5+)
1. **방향 선택**: 시계/반시계 중 어느 방향이 소행성을 막을 수 있는가
2. **우선순위**: 여러 소행성 중 어느 것을 먼저 막을 것인가
3. **콤보 유지**: 콤보 타이머 내에 다음 튕겨내기 성공할 수 있는가
4. **파워업 수집**: 궤도 위 파워업을 방패 끝으로 수집할 타이밍
5. **다단 적 대응**: HP 2-3 소행성은 여러 번 맞춰야 하므로 궤도 위치 예측
6. **리스크 관리**: 무적 시간 활용, 체력 관리

## 프리미엄 유료 앱 근거
- **깔끔한 경험**: 광고 없이 몰입 유지. 원형 궤도 + 우주 테마의 시각적 일관성
- **즉각적 재미**: 첫 탭부터 만족감. 학습 비용 0
- **깊은 숙련도**: 콤보 시스템 + 다중 소행성 대응은 수십 시간 플레이에도 스킬 성장 느낌
- **$2.99 가치**: 하이스코어 경쟁, 난이도 곡선, 시각 효과 품질

## 리텐션 메카닉

### 세션 리텐션 (한 판 내)
- 콤보 배율 유지 압박 → "한 번 더 튕겨내면 x5!"
- 난이도 점진 상승 → 항상 다음 벽에 도전
- 슬로우 모션 피격 → 극적 긴장감

### 일간 리텐션
- 베스트 스코어 갱신 욕구
- "오늘은 콤보 x10 달성하겠다"
- 레벨 10+ 도달 도전

### 장기 리텐션
- 완전한 숙련 (소행성 패턴 예측)
- 하이스코어 자기경쟁
- 무피격 클리어 도전

## 기술 아키텍처
- **불변 상태**: GameState data class → 순수 함수로 업데이트
- **60fps 게임 루프**: withFrameNanos + 델타 타임
- **충돌 검출**: 원형 궤도 기반 각도 범위 체크
- **파티클 시스템**: 최대 200개 파티클, 프레임 드롭 없음
- **절차적 생성**: 난이도 기반 소행성 종류/속도/수량 조절

## 확장 가능성
- 다양한 행성 스킨
- 이중 방패 모드
- 보스 소행성 (특수 패턴)
- 리더보드
