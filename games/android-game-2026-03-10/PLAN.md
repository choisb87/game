# Getaway: Night Heist — 게임 디자인 플랜

## 장르 선정 과정

### 후보 3개 내부 검토

| # | 장르 | 경찰/도둑 역할 구분 | 판단 깊이 | 첫 30초 긴장감 | 반복 동기 |
|---|------|---------------------|-----------|----------------|-----------|
| 1 | **탑다운 스텔스 추격** | 도둑=플레이어, 경찰=AI순찰 시야콘 | 루트 계획, 리스크/리워드, 타이밍 | 즉시 (경찰이 눈앞에서 순찰) | 패턴 학습, 레벨 진행, 점수 |
| 2 | 사이드스크롤 파쿠르 도주 | 도둑=달리기, 경찰=뒤쫓기 | 점프/슬라이드 타이밍 (얕음) | 보통 | 장애물 패턴 |
| 3 | 턴제 전술 추격전 | 교대 이동 | 높지만 느림 | 낮음 (턴제라 긴장감 약함) | 퍼즐적 만족감 |

### 선택: 1번 — 탑다운 스텔스 추격

**이유**: 경찰과 도둑의 역할이 가장 명확하고, 매 순간 "들킬까 말까"의 긴장감이 코어루프를 관통한다. 판단 요소(루트 선택, 은신 타이밍, 보석 수집 순서)가 단순 반사신경을 넘어선다.

## 핵심 재미 루프

### 코어 루프 (1분 사이클)
```
보석 발견 → 경찰 시야 파악 → 루트 결정 → 이동 → [들킴?]
  ├─ 안 들킴 → 보석 획득 → 콤보 유지? → 다음 보석으로
  ├─ 들킴 → 파워업 사용? → 도주 → 은신처 숨기 → 재시도
  └─ 파워업 발견 → 수집할까? 보석 먼저? → 전략적 판단
전체 보석 수집 → 탈출구 개방 → 탈출 루트 결정 → 보너스 계산 → 레벨 클리어
```

### 첫 30초 긴장감 설계
1. **0-3초**: 레벨 인트로 — 카메라가 탈출구에서 시작점까지 패닝. 맵 구조와 경찰 배치를 한눈에 파악
2. **3-10초**: 첫 보석이 가까이 있지만 경찰 순찰 경로 위. 타이밍을 봐야 함
3. **10-20초**: 첫 보석 획득 → 골드 파티클 + "+100" 텍스트 → 파워업 발견 → 수집할까?
4. **20-30초**: 콤보를 이어갈까 안전하게 갈까 → 경찰 시야에 걸릴 뻔 → 빨간 경고 → 긴장감 최고조

### 판단 요소 (최소 4가지)
1. **루트 선택**: 어떤 순서로 보석을 수집할 것인가
2. **스닉 vs 러닝**: 조이스틱 드래그 거리로 속도-은밀도 트레이드오프
3. **파워업 전략**: SMOKE/SPEED/GHOST 중 상황에 맞는 것 수집
4. **콤보 유지**: 빠르게 연속 수집하면 고배율, 하지만 리스크 증가
5. **은신 타이밍**: 언제 숨고 언제 움직일 것인가

### 왜 이전안(Gravity Pulse)보다 나은가

| 항목 | Gravity Pulse | Getaway: Night Heist |
|------|---------------|----------------------|
| 판단 깊이 | 탭 타이밍 1가지 | 루트/은신/수집순서/파워업/콤보 다층 판단 |
| 역할 체감 | 없음 (추상적 오브) | 도둑으로서 긴장감, 경찰 AI와 두뇌싸움 |
| 서사 몰입 | 없음 | "야간 도주"라는 명확한 판타지 |
| 레벨 구조 | 무한 스크롤 (단조) | 절차적 생성 레벨, 점진적 난이도 |
| 감정 곡선 | 긴장→죽음 단조 반복 | 탐색→긴장→안도→콤보 흥분→탈출 풍부한 감정 곡선 |
| 반복 동기 | 하이스코어만 | 레벨 진행 + 콤보 + 파워업 + 보너스 + 통계 |
| 첫인상 | 바로 게임 시작 | 레벨 인트로 → 맵 파악 → 전략 수립 |
| 시각 피드백 | 점으로 표시 | 실제 텍스트, 화면 흔들림, 콤보 표시, 점수 분해 |

## 유료앱으로서 가치 포인트

### 가격: $3.99

1. **광고 없는 몰입**: 긴장감이 핵심인 스텔스 게임에서 광고는 치명적. 프리미엄으로 끊김 없는 경험 보장
2. **절차적 레벨 생성**: 매번 다른 맵 → 무한 콘텐츠. "몇 판만 하면 끝"이 아님
3. **전략적 깊이**: 5가지 이상 판단 요소 (루트/스닉/파워업/콤보/은신) → 유료 가치 체감
4. **파워업 + 콤보 시스템**: 단순 수집이 아닌 전략적 아이템 활용과 리스크-리워드 콤보
5. **완전한 게임**: IAP 없음, 잠금 콘텐츠 없음, 에너지 시스템 없음. 결제 = 풀 게임
6. **오프라인 플레이**: 인터넷 불필요, 작은 용량, 배터리 효율적
7. **프리미엄 연출**: 레벨 인트로, 화면 흔들림, 콤보 피드백, 점수 분해 화면 — 유료앱 퀄리티

### 시장 포지셔닝

- **"이걸로 돈 받아?"가 아닌 "이 가격에 이 깊이?"로 전환**

## 게임 메카닉 상세

### 플레이어 (도둑)
- 가상 조이스틱으로 전방향 이동
- **이중 속도**: 짧은 드래그 = 스닉(1.8), 긴 드래그 = 달리기(3.8)
- 기본 속도 > 순찰 경찰, < 추격 경찰
- 은신처(파란 타일)에서 정지 시 자동 은신
- 은신 중에는 경찰 시야에 걸리지 않음

### 파워업 시스템
- **SMOKE (보라)**: 반경 5타일 내 경찰 3초 기절. 위급 상황 탈출용
- **SPEED (시안)**: 5초간 이동속도 1.5배. 빠른 수집 + 도주용
- **GHOST (연보라)**: 3초간 완전 투명. 위험 구간 돌파용
- 레벨당 1~3개 배치 (레벨 높을수록 많음)
- 마지막 사용한 파워업만 활성 (중첩 불가)

### 콤보 시스템
- 4초 안에 연속 보석 수집 시 콤보 카운트 증가
- 배율: 1x → 1.5x → 2x → 2.5x ...
- 콤보 중 파티클 색상/수량 증가
- 레벨 클리어 시 최대 콤보 × 200 보너스
- 시각: 화면 상단에 "x3 COMBO!" 텍스트 + 타이머 바

### 경찰 AI
- **순찰 (PATROL)**: 정해진 루트를 따라 이동, 시야콘 표시
- **경계 (ALERT)**: 플레이어 발견 시 0.6초 경계 → 방향 전환
- **추격 (CHASE)**: 플레이어를 직접 추격, 속도 증가
- **수색 (SEARCH)**: 플레이어를 놓치면 마지막 위치에서 주위 수색
- **기절**: SMOKE 파워업 효과. 3초간 무력화
- 레벨이 높아질수록: 더 빠르고, 시야 넓고, 수가 많음

### 레벨 시스템
- BSP 기반 절차적 맵 생성 (방 + 복도)
- **레벨 인트로**: 카메라가 탈출구→시작점으로 패닝 (레벨1은 3.5초 + 튜토리얼)
- 보석 수: 3개(레벨1) → 최대 8개
- 경찰 수: 2명(레벨1) → 최대 6명
- 파워업: 1개(레벨1) → 최대 3개
- 증원 타이머: 45초(레벨1) → 최소 25초
- 은신처 배치, 라이프 3개

### 점수 체계
- 보석 수집: 100 × 레벨 × 콤보배율
- 시간 보너스: 남은 시간 × 10
- 스텔스 보너스: 한 번도 안 들키면 +500
- 콤보 보너스: 최대 콤보 × 200
- **레벨 클리어 시 점수 분해 화면 표시**
- 누적 점수로 최고 기록 경쟁

## 기술 설계

### 아키텍처
- 불변 `GameState` data class → 순수 함수 업데이트
- Compose Canvas + Android Native Canvas (텍스트) 하이브리드 렌더링
- `LaunchedEffect` + `awaitFrame`으로 60fps 게임 루프
- SharedPreferences로 최고 점수 + 통계 저장

### 파일 구조
```
game/
  GameState.kt    — 모든 데이터 클래스, 파워업/콤보 상태 포함
  GameMap.kt      — BSP 절차적 맵 생성
  GameLogic.kt    — 게임 업데이트, AI, 파워업, 콤보, 인트로
  GameRenderer.kt — Canvas 렌더링, 텍스트 HUD, 연출, 오버레이
  GameScreen.kt   — Compose 화면 + 게임 루프 + 통계 저장
  MenuScreen.kt   — 메인 메뉴 (브랜딩, 기록, 조작법)
```

### 렌더링 특징
- 카메라 추적 (플레이어 중심 부드러운 팔로우)
- **화면 흔들림** (체포 시, 증원 도착 시)
- 경찰 시야콘 시각화 (상태별 색상 변화)
- 미니맵 (우하단, 파워업 포함)
- 위험 경고 오버레이 (화면 가장자리 빨간 바이넷)
- **텍스트 HUD**: 레벨, 라이프, 보석 카운트, 점수, 증원 타이머, 스닉 표시
- **콤보 표시**: 화면 상단 콤보 텍스트 + 타이머 바
- **파워업 표시**: 좌하단 활성 파워업 이름 + 잔여 시간 바
- **레벨 인트로 오버레이**: 레벨 번호 + 레벨별 자막 + 튜토리얼(레벨1)
- **점수 분해 오버레이**: 레벨 클리어 시 각 보너스 항목별 표시
- 파워업 아이템: 다이아몬드 아이콘 + 타입별 색상 + 맥동 글로우
- 가상 조이스틱 (스닉존 시각 표시)
