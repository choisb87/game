# Market Benchmark: 중력 반전 러너 장르 분석

## 1. 상위 차트 레퍼런스

### Geometry Dash (RobTop Games)
- **장르**: 리듬 기반 플랫포머
- **핵심**: 원터치 점프, 음악 동기화, 극한 난이도
- **채택할 점**:
  - 원터치 조작의 순수함 → 우리도 탭 하나로 모든 것을 해결
  - 시각적 피드백의 화려함 → 네온 글로우 + 파티클 이펙트
  - 즉시 재시도 → 사망 후 1탭 리스타트
- **피할 점**:
  - 레벨 기반 구조 (콘텐츠 소진 문제) → 절차적 생성으로 해결
  - IAP + 광고 혼합 모델 → 순수 유료

### VVVVVV (Terry Cavanagh)
- **장르**: 중력 반전 플랫포머
- **핵심**: 중력 반전 메커니즘의 원조, 레트로 미학
- **채택할 점**:
  - 중력 반전이 주는 독특한 공간 인식 → 코어 메커니즘 그대로
  - 미니멀한 조작이 만드는 깊이 → 탭 하나에 모든 전략 집중
- **피할 점**:
  - 고정 맵 (리플레이성 제한) → 절차적 생성
  - PC 중심 UX → 모바일 원터치 최적화

### Flappy Bird (Dong Nguyen)
- **장르**: 원터치 장애물 회피
- **핵심**: 극단적 단순함, 중독성, 하이스코어 경쟁
- **채택할 점**:
  - 3초 학습 곡선 → 첫 탭으로 바로 이해
  - "한 번 더" 중독성 → 즉시 재시도 + 최고 기록 표시
  - 짧은 세션 → 30초~3분
- **피할 점**:
  - 단조로운 비주얼 → 네온 미학 + 존 변화
  - 보상 체계 부재 → 콤보 + 크리스탈 + 존 시스템

### Polara (Hope This Works Games)
- **장르**: 컬러 스위칭 러너
- **핵심**: 두 상태 사이 전환하며 장애물 통과
- **채택할 점**:
  - 이진 상태 전환의 전략적 깊이 → 중력 상/하 = 두 경로 선택
  - 시각적 명확성 (어떤 상태가 안전한지 즉시 파악)
- **피할 점**:
  - 스테이지 기반 → 무한 러너

### Alto's Odyssey (Snowman)
- **장르**: 무한 러너 (스노보드)
- **핵심**: 미학적 완성도, 명상적 게임플레이
- **채택할 점**:
  - 프리미엄 비주얼 퀄리티 → 네온 글로우, 패럴랙스 배경
  - 유료 모델 성공 사례 → 광고 없는 프리미엄 경험
- **피할 점**:
  - 느린 템포 → 더 긴장감 있는 속도 에스컬레이션

## 2. 시장 포지셔닝

```
         복잡한 조작
              |
   퍼즐       |    액션 RPG
   게임       |
              |
느림 ─────────┼───────── 빠름
              |
   명상적     |  ★ Void Runner
   러너       |    (원터치 + 빠름 + 깊이)
              |
         단순한 조작
```

### Void Runner의 포지션
- **조작**: 극도로 단순 (원터치)
- **속도**: 빠름 (에스컬레이팅)
- **깊이**: 중간~높음 (5가지 판단 레이어)
- **세션**: 초단기 (30초~3분)
- **비주얼**: 프리미엄 네온

## 3. 채택/회피 요약

### 채택 (DO)
| 요소 | 레퍼런스 | 적용 방식 |
|------|----------|-----------|
| 원터치 순수함 | Flappy Bird, Geometry Dash | 탭 하나로 중력 반전 |
| 즉시 재시도 | Flappy Bird | 사망 → 1탭 리스타트 |
| 프리미엄 비주얼 | Alto's Odyssey | 네온 글로우 + 파티클 |
| 콤보 보상 | Geometry Dash | 연속 수집 배율 시스템 |
| 절차적 생성 | 로그라이크 장르 | 매 런 다른 장애물 배치 |
| 유료 모델 | Alto, VVVVVV | $2.99, 광고/IAP 없음 |

### 회피 (DON'T)
| 요소 | 이유 |
|------|------|
| 레벨/스테이지 잠금 | 리플레이성 제한, IAP 유도 함정 |
| 광고 삽입 | 리듬 파괴, 재시도 의욕 소멸 |
| 복잡한 커스터마이징 | 코어 루프 희석, 첫 경험 지연 |
| 에너지/하트 시스템 | 유료 앱에 페이월 = 이중 과금 인식 |
| 소셜 기능 강제 | 오프라인 가치 훼손 |
| 튜토리얼 강제 | 원터치라 불필요, 자연 학습 |

## 4. 경쟁 우위

1. **중력 반전 + 자동 러너**: VVVVVV의 메커니즘을 모바일 원터치로 재해석
2. **콤보 + 존 시스템**: Flappy Bird의 단순함에 깊이 추가
3. **절차적 무한**: Geometry Dash의 레벨 소진 문제 해결
4. **순수 유료**: 광고/IAP 없는 깨끗한 경험
5. **30초 훅**: 첫 탭부터 마지막 사망까지 끊김 없는 루프
