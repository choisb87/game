# Market Benchmark: Flame Dash

## 경쟁작 분석

### 1. Doodle Jump
- **장르**: 수직 점프 / Endless Climber
- **핵심**: 틸트 조작으로 플랫폼 밟아 올라감
- **강점**: 직관적, 중독적, 오랜 레거시
- **약점**: 틸트 조작은 공공장소에서 불편, 비주얼 단순
- **참고점**: 수직 상승 = 본능적 목표 → 채택

### 2. Flappy Bird
- **장르**: 원터치 아케이드
- **핵심**: 탭으로 날개짓, 파이프 사이 통과
- **강점**: 극도로 단순한 조작, 순간적 긴장
- **약점**: 랜덤성 부족, 단조로움
- **참고점**: 원터치 → 채택, 단조로움 → 회피 (다양한 플랫폼 타입으로 해결)

### 3. Geometry Dash
- **장르**: 리듬 액션 플랫포머
- **핵심**: 탭으로 점프, 장애물 패턴 암기
- **강점**: 음악 싱크, 레벨 디자인, 스킬 표현
- **약점**: 암기 기반 → 첫 플레이 좌절, $2 유료
- **참고점**: 높은 폴리시 + 유료 모델 가능 → 채택, 암기 기반 → 회피 (프로시저럴로 대체)

### 4. Helix Jump
- **장르**: 캐주얼 하강 아케이드
- **핵심**: 나선형 탑을 통과하며 내려감
- **강점**: 만족스러운 물리, 단순 조작
- **약점**: 광고 폭탄, 깊이 부족
- **참고점**: 만족스러운 물리감 → 채택, 광고 모델 → 회피

### 5. Alto's Odyssey
- **장르**: 엔드리스 러너 / 아트 게임
- **핵심**: 스노보드 활강, 공중 트릭
- **강점**: 미학적 완성도, 사운드스케이프, 프리미엄 품질
- **약점**: 액션 빈도 낮음 (명상적)
- **참고점**: 프리미엄 품질 기준 → 채택, 높은 액션 빈도 유지

## 포지셔닝 맵

```
        깊은 전략
            |
  Geometry  |
   Dash     |
            |
단순 ———————+——————— 복잡
  조작      |      조작
   Flappy   | ★ Flame Dash
   Bird     |
            |
   Doodle   |
   Jump     |
        즉각적 반응
```

**Flame Dash 위치**: 단순 조작(좌/우 탭) + 즉각적 반응 + 중간 전략 깊이

## 채택할 요소

| 요소 | 출처 | 적용 |
|------|------|------|
| 수직 상승 목표 | Doodle Jump | 끝없이 올라가는 구조 |
| 원터치 조작 | Flappy Bird | 화면 좌/우 탭만으로 플레이 |
| 비주얼 폴리시 | Geometry Dash | 파티클, 화면 효과, 색상 설계 |
| 프리미엄 가격 | Geometry Dash | $2.99, 광고 없음 |
| 프로시저럴 맵 | Doodle Jump | 매번 새로운 경험 |
| 다양한 플랫폼 | Doodle Jump | 4종 플랫폼 타입 |
| 콤보 시스템 | Alto's Odyssey | 보석 연속 수집 보너스 |

## 회피할 요소

| 요소 | 이유 |
|------|------|
| 광고 삽입 | 30초~3분 세션에 광고는 치명적 |
| 틸트 조작 | 공공장소 사용 불편, 접근성 낮음 |
| 레벨 암기 | 첫 플레이 좌절감, 반복 강제 |
| 에너지/라이프 시스템 | 유료 게임에 맞지 않음 |
| 과도한 IAP | 유료 앱 + IAP = 부정적 리뷰 |
| 소셜 강제 | 순수 솔로 경험에 집중 |

## 경쟁 우위

1. **대시 메커니즘**: 단순 점프가 아닌 대시 — 속도감과 전략성 동시 제공
2. **화염 추적**: 단순 낙사가 아닌 용암 추적 — 항상 긴장감 유지
3. **화면 랩**: 화면 좌우 끝이 연결 — 숏컷 전략이라는 독특한 레이어
4. **시각적 폴리시**: 캐주얼 클리커와 차별화된 파티클/이펙트
5. **프리미엄 경험**: 광고/IAP 없는 깨끗한 경험 → 리뷰 점수 우위
