# 시장 벤치마크 분석

## 참고 타이틀 (Top Chart References)

### 1. Stack (Ketchapp)
- **장르**: 타이밍 기반 스태킹
- **강점**: 극도로 단순한 조작, 즉각적 이해, 중독성 있는 루프
- **약점**: 광고 과다, 시각적 단조로움, 장기 목표 부재
- **채택할 점**: 원탭 메커닉, 점진적 난이도 커브
- **피할 점**: 인터스티셜 광고 폭격, 최소한의 시각 피드백

### 2. 1010! / Tetris
- **장르**: 퍼즐/스태킹
- **강점**: 전략적 깊이, 강한 리텐션, 글로벌 인지도
- **약점**: 학습 곡선 존재, 세션 길이 김
- **채택할 점**: "쌓기"라는 보편적 매력
- **피할 점**: 복잡한 규칙, 긴 세션 (캐주얼에 부적합)

### 3. Flappy Bird
- **장르**: 원탭 아케이드
- **강점**: 극단적 단순성, 바이럴 요소, "한 판 더" 중독성
- **약점**: 프러스트레이션 과다, 단조로운 비주얼
- **채택할 점**: 즉사 → 즉시 재시작 루프, 점수 경쟁 동기
- **피할 점**: 과도한 난이도 스파이크

### 4. Alto's Odyssey
- **장르**: 러너/아케이드
- **강점**: 시각적 아름다움, 명상적 분위기, 프리미엄 품질감
- **약점**: 아케이드 긴장감 부족
- **채택할 점**: 시각적 분위기 (밤하늘, 색감), 프리미엄 질감
- **피할 점**: 느슨한 게임 루프

### 5. Crossy Road
- **장르**: 원탭 아케이드
- **강점**: 매력적 캐릭터, 무한 리플레이, 적절한 수익화
- **약점**: 가챠 요소
- **채택할 점**: 짧은 세션, 높은 리플레이성
- **피할 점**: 캐릭터 언락 가챠

## 시장 포지셔닝

### 차별화 전략
| 요소 | 경쟁작 | Sky Stacker |
|------|--------|-------------|
| 수익화 | 광고 기반 F2P | 유료 프리미엄 |
| 세션 길이 | 30초~5분 | 30초~2분 |
| 시각 품질 | 미니멀/저가 | 밤하늘 + 파티클 연출 |
| 보상 체계 | 광고 시청 보상 | 스킬 기반 PERFECT 콤보 |
| 조작 | 원탭 | 원탭 |

### 타겟 유저
- **핵심**: 짧은 시간에 집중적 재미를 원하는 캐주얼 게이머
- **확장**: 광고 없는 깔끔한 경험을 선호하는 프리미엄 유저
- **연령대**: 전 연령 (PEGI 3 / ESRB E)

### 채택 요소 요약
1. 원탭 즉시 플레이 (Stack, Flappy Bird)
2. 시각적 분위기와 품질감 (Alto's Odyssey)
3. 스킬 보상 시스템 PERFECT (리듬게임 영향)
4. 즉사 즉시 재시작 (Flappy Bird)
5. 프리미엄 유료 모델 (Monument Valley, Threes!)

### 피할 요소 요약
1. 인터스티셜 광고 (Ketchapp 류)
2. 가챠/랜덤박스 (Crossy Road)
3. 에너지 시스템/타임게이팅
4. 소셜 강제 (초대하기 보상)
5. 과도한 IAP 유도
