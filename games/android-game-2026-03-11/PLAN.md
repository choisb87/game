# Sky Stacker - 게임 설계 문서

## 핵심 재미 루프 (Fun Loop)

### 30초 훅
1. **즉각적 이해** (0-3초): 블록이 움직이는 것을 보고 탭 → 바로 결과 확인
2. **첫 성공 경험** (3-10초): 2-3개 성공적으로 쌓으며 "나 잘하네" 느낌
3. **긴장감 도입** (10-20초): 속도 증가 + 블록 좁아짐 → 집중력 요구
4. **PERFECT 발견** (20-30초): 정확히 맞추면 골든 이펙트 + 콤보 → "다시 해야지"

### 핵심 메커닉
- **원탭 조작**: 진입 장벽 제로. 누구나 3초 안에 플레이 가능
- **점진적 난이도**: 속도↑ + 블록 폭↓ = 자연스러운 텐션 커브
- **PERFECT 시스템**: 스킬 보상. 정확한 타이밍 → 블록 폭 유지 + 콤보 점수
- **시각 피드백**: 파티클, 화면 흔들림, 플래시 → 액션에 대한 즉각 반응

## 리텐션 설계

### 단기 리텐션 (세션 내)
- "한 판만 더" 유도: 즉사 → 바로 재시작 (0.5초)
- 점수 경쟁: 자기 최고 기록 갱신 욕구
- PERFECT 콤보 체이닝: 연속 정확 배치의 만족감

### 중기 리텐션 (1-7일)
- 개인 최고 기록 지속적 갱신 도전
- 콤보 마스터리: 연속 PERFECT의 숙련 곡선
- 속도 적응: 고속 구간 돌파 성취감

### 장기 리텐션
- 기록 갱신의 끝없는 도전
- 짧은 세션(30초-2분)으로 틈틈이 플레이

## 유료 앱 전략 (Premium Paid)

### 광고 없는 이유
- **몰입 파괴 방지**: 타이밍 게임에서 광고 인터스티셜은 리듬 파괴
- **공정한 경험**: 모든 유저가 동일한 경험. P2W 없음
- **깔끔한 UX**: 배너 없이 전체화면 몰입
- **신뢰 형성**: "개발자가 내 시간을 존중한다"는 인식

### 가격 정당성
- 즉각적 재미: 설치 후 3초 만에 플레이 가능
- 무한 리플레이: 콘텐츠 소비형이 아닌 스킬 기반
- 오프라인 플레이: 인터넷 불필요
- 배터리 효율: 경량 렌더링, 과도한 리소스 사용 없음

### 가격대
- **$1.99 ~ $2.99** 권장
- 커피 한 잔보다 저렴, 수십 시간 플레이 가능
- "완성된 게임" 느낌의 프리미엄 퀄리티

## 기술 설계

### 아키텍처
- **순수 상태 기반**: GameState 불변 데이터 → 순수 함수로 업데이트
- **단방향 데이터 흐름**: Input → Logic → State → Render
- **Compose Canvas**: 네이티브 성능, 별도 엔진 종속성 없음

### 성능 목표
- 60fps 유지 (16ms 프레임 버짓)
- 메모리 할당 최소화 (파티클 리스트 재사용)
- GC 압박 최소화
